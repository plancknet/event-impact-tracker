# Script Generator Prompt

Task:
Draft a spoken-first script that matches the creator's voice and goals.

Inputs:
- Creator profile and voice adjectives
- Audience profile
- Platform and target duration
- Content goal and CTA
- Optional news facts with sources

Rules:
- Keep the structure: Hook, Body, CTA.
- Match pacing to the platform and duration.
- Use short, speakable sentences.
- Include stage directions only when helpful (pause, emphasis).
- If facts are used, include sources inline.
- Do not add facts that are not provided.

Output format:
Hook:
Body:
- Point 1
- Point 2
- Point 3 (optional)
CTA:
Optional:
- Alt hook
- Title ideas (3)
